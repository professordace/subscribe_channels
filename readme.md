<h2>Подписные каналы</h2>

<h3>Требования</h3>
PHP >= 7.2,  
Mysql  

<h3>Установка</h3>
Для установки, выполните следующие пункты в папке проекта:

1) composer install
2) .env прописываем данные для БД
3) php artisan migrate
4) php artisan db:seed
